# Input

**Keyboard and Mouse:**

WASD - Move

Space - Jump

Mouse - Aim

Left Click - Shoot

Right Click - Aim Down Sights

Hold Shift - Sprint

CTRL - Slide

**Gamepad:**

Left Stick - Move

Left Bumper - Jump (trust me bro)

Right Stick - Aim

Right Trigger - Shoot

Left Trigger - Aim down Sights

Left Stick Button - Sprint

Right Stick Button - Slide
